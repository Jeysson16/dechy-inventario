# Banners Promocionales

## /hero — Hero Banner (Banner de Campaña)

Banners de ancho completo para campañas grandes y storytelling visual.

- **Resolución:** 1440 × 420 px
- **Formato:** JPG o WebP
- **Archivos:** hero-1.jpg, hero-2.jpg, ...

## /split — Split Banner (Banner Partido)

Banners divididos: texto a la izquierda + imagen del producto a la derecha.

- **Resolución total:** 1200 × 400 px
- **Lado imagen:** 600 × 400 px
- **Archivos:** split-1.jpg, split-2.jpg, ...

## /promo — Promotional Cards (Tarjetas Promocionales)

Tarjetas para destacar ofertas específicas o categorías.

- **Resolución:** 400 × 300 px
- **Formato:** JPG o WebP
- **Archivos:** promo-1.jpg, promo-2.jpg, ...
