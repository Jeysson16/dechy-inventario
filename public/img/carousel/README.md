# Imágenes del Carrusel Hero

Coloca aquí las imágenes del carrusel de la página de inicio.

## Formato recomendado

- **Resolución:** 1440 × 560 px (mínimo 1200 × 480 px)
- **Formato:** JPG o WebP (WebP para mejor rendimiento)
- **Peso máximo:** 300 KB por imagen

## Nomenclatura

- slide-1.jpg → Primera diapositiva
- slide-2.jpg → Segunda diapositiva
- slide-3.jpg → Tercera diapositiva
- _(agregar más según necesidad)_

## Configuración

Edita el archivo `src/shop/components/HeroCarousel.jsx` para:

- Cambiar los textos de cada slide (título, subtítulo, CTA)
- Agregar o quitar slides
- Modificar el intervalo de cambio (por defecto 4 segundos)
